A Regra dos 80/20: A IA vai te levar aos primeiros 80% do projeto em tempo recorde, mas os últimos 20% (para fazer o sistema não quebrar, lidar com erros e ser seguro) vão tomar 90% do seu tempo de desenvolvimento

Como fazer do jeito certo (Usando sua vantagem de Arquiteto):

Para que o Claude Code crie o seu projeto facilmente, você deve adotar este fluxo de trabalho:
Crie o Contexto Global (O Arquivo agents.md): Antes de gerar qualquer código, crie um arquivo de texto no seu projeto (como um agents.md ou .cursorrules) definindo a stack (FastAPI, React, Supabase), a arquitetura e as regras de estilo

Versionamento Constante: Faça commits e use branches no Git com muita frequência. A IA adora fazer grandes refatorações que quebram o aplicativo inteiro. Você precisa conseguir reverter para um estado funcional facilmente
